# 大一班会 · 文字冒险游戏 — 操作指南

本指南适用于在教室投影环境下，使用 Streamlit 文字冒险游戏进行班会互动。

---

## 一、这是什么？

这是一款面向大一班会的**文字冒险互动网页**。AI 担任游戏主持人（GM），围绕以下主题推进剧情：

- **期末诚信考试教育**（作弊、抄袭等违纪后果）
- **正确成长观**（学业压力、挂科焦虑、脚踏实地）

全班可以通过点击选项或输入自定义行动，共同决定剧情走向。

---

## 二、运行前准备

### 1. 电脑环境要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows / macOS 均可 |
| Python | 3.10 或以上 |
| 网络 | 能访问 AI 接口（DeepSeek、OpenAI 等） |
| 文件 | 本目录下的 `app.py` |

### 2. 安装依赖（首次使用）

打开终端（Windows 为 PowerShell 或 CMD），进入项目目录，执行：

```bash
pip install streamlit openai
```

若下载较慢，可使用国内镜像：

```bash
pip install streamlit openai -i https://pypi.tuna.tsinghua.edu.cn/simple
```

### 3. 准备接口密钥

向 DeepSeek、OpenAI 等平台申请 API 密钥，班会开始前填好即可。**请勿将密钥泄露或公开分享。**

---

## 三、如何启动

1. 打开终端，进入 `app.py` 所在目录
2. 执行：

```bash
streamlit run app.py
```

3. 浏览器会自动打开页面（地址一般为 `http://127.9.30.1:8501`）
4. 将浏览器窗口**全屏**（按 `F11`），连接投影仪

> 建议：班会前在**实际使用的电脑**上完整试跑一遍，确认网络和接口正常。

## 五、侧边栏接口配置

| 配置项 | 说明 | 常用示例 |
|--------|------|----------|
| **接口密钥** | AI 平台的 API Key | 在平台控制台获取 |
| **接口地址** | API 服务地址 | `https://api.deepseek.com` |
| **模型名称** | 使用的模型 | `deepseek-chat` |

### 使用 OpenAI 时

| 配置项 | 填写内容 |
|--------|----------|
| 接口地址 | `https://api.openai.com/v1` |
| 模型名称 | `gpt-4o` 或 `gpt-4o-mini` |

填写**接口密钥**后，页面会自动加载 GM 开场白，无需额外点击开始。

## 六、班会现场操作流程

### 第一步：会前配置（建议投影前完成）

1. 启动程序：`streamlit run app.py`
2. 在**侧边栏**填写接口密钥、接口地址、模型名称
3. 确认上方「剧情记录」出现 GM 开场白
4. 浏览器全屏，检查投影字号是否清晰
5. **收起或关闭侧边栏**（点击页面左上角 `>` 箭头），避免密钥被投影到屏幕上

### 第二步：开场介绍（约 1 分钟）

向全班说明：

- 这是一个关于**诚信考试**和**正确成长观**的文字冒险
- 每轮 GM 会给出剧情和 A、B、C 三个选项
- 全班可以**投票选一个**，或在下方**输入商定的自定义行动**
- 由操作员（你或同学）在电脑上点击或提交

### 第三步：每轮互动

1. **阅读剧情**：操作员或主持人朗读/概括 GM 给出的剧情和 A/B/C 含义
2. **全班讨论**：快速商量选哪个，或想出自定义方案
3. **提交选择**（二选一）：
   - 点击 **「选项 A / B / C」** 按钮
   - 或在文本框输入自定义内容，点击 **「提交自定义选择」**
4. **等待加载**：页面显示「剧情推进中...」，约几秒到十几秒
5. **查看新剧情**：上方历史区会追加「班级选择」和 GM 新回复
6. 重复以上步骤，直到本局结束

### 第四步：本局结束

当 GM 判定游戏结束时，页面会提示：

> 本局已结束，可在侧边栏点击「重新开始游戏」。

此时选项按钮和自定义输入会被禁用。若要再来一局：

1. 打开侧边栏
2. 点击 **「重新开始游戏」**
3. 自动获取新的开场剧情

## 七、两种选择方式对比

| 方式 | 操作 | 适用场景 |
|------|------|----------|
| **选项 A/B/C** | 直接点击对应按钮 | 全班快速投票，选 GM 预设选项 |
| **自定义选择** | 在文本框输入内容 → 点击「提交自定义选择」 | 全班讨论后提出独特方案 |

自定义示例：

- 「向老师主动坦白并申请补考」
- 「拒绝室友的作弊请求，一起复习」
- 「制定每周学习计划，找学长请教」
- 「学习学长的[测控](https://github.com/ucxn/ZTE-Stat_Max/blob/main/new.user.js)算法[程序](https://github.com/ucxn/Bro-Stat/raw/refs/heads/main/Huawei-1.user.js)，提升自我」

## 八、班会互动建议

1. **控制节奏**：每轮讨论 1～2 分钟，避免拖沓
2. **主持人引导**：帮全班理解 A/B/C 各自代表什么再投票
3. **鼓励自定义**：有想法时优先用自定义输入，互动感更强
4. **操作员固定一人**：避免多人同时操作电脑
5. **历史区可回看**：上方滚动区域保留全部对话，方便总结时回顾
6. **刷新页面**：同一浏览器标签页内刷新，对话历史一般会保留；若丢失，点「重新开始游戏」即可

## 九、常见问题

### Q1：页面提示「请先在侧边栏填写接口密钥以开始游戏」

**原因**：未填写接口密钥。  
**解决**：打开左侧侧边栏，填写接口密钥。

### Q2：提示「接口请求失败」

**可能原因**：

- 接口密钥错误或已过期
- 网络无法访问 AI 接口
- 接口地址或模型名称填写错误

**解决**：逐项检查侧边栏配置；换用手机热点测试是否为网络问题。

### Q3：点击选项后没有反应 / 一直加载

**原因**：网络慢或接口繁忙。  
**解决**：等待 10～20 秒；若仍失败，查看页面红色错误提示，检查网络后重试。

### Q4：用了一次自定义选择后就不能再选了

**原因**：旧版本或未升级误判游戏结束，当前版本已修复。  
**解决**：检查本地游戏版本，确保使用最新 `app.py`；若已卡住，侧边栏点「重新开始游戏」。

### Q5：教室电脑没有 Python 怎么办？

**方案**：使用学长的专线（例如汉口独立城域网、直通电信物理BRAS）运行RDP协议，班级直接使用mstsc。

### Q6：如何停止程序？

在运行 `streamlit run app.py` 的终端窗口按 `Ctrl + C`，或直接关闭窗口。
